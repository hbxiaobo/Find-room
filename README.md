#### 海鸥租房
* 涵盖技术
	1. html/html5
	2. css/css3
	3. javascript,jquery
	4. ajax
	5. nodejs静态服务器搭建
* 下载
  - `git clone https://github.com/hbxiaobo/Find-room.git`
* 进入
  - `cd Find-room`
* 安装依赖
  - `npm install`
* 开发环境
  - 启动
    - `gulp`
  - 访问
    - `http://127.0.0.1:9800`
* 部署环境
  - 构建
    - `gulp install`
  - 启动
    - `node app.js`
  - 访问
    - `http://127.0.0.1:16084`
